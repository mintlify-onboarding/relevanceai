## Sentiment

Extract sentiment from your text.

Python

`text_fields = ["reviews"]
ds.extract_sentiment(text_fields=text_fields)
`

### Sentiment with HuggingFace

You can pass through any sentiment analysis model from HuggingFace through the `model_name` to use that model instead.

Python

`ds.extract_sentiment(text_fields=text_fields, model_name="cardiffnlp/twitter-roberta-base-sentiment-latest")
`

For enterprise we provide a model trained on millions of text.
